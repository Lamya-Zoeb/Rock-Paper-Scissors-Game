## Instructions
This is a Rock, Paper, Scissors game!! 
You have to use either Rock, Paper, or Scissors!
Rock beats the Scissors
Paper beats the Rock
Scissors beats the paper
You have 3 lives! If you lose all your lives you lose :(
ENJOY! :)
